## How to Use / 如何使用

| Method / 方式 | Example / 示例 |
|------|------|
| **@mention** (CodeBuddy, Antigravity) | `@tdd-workflow Help me write tests` |
| **/slash command** (some tools) | `/tdd-workflow Help me write tests` |
| **Natural language** (AI auto-matches) | "Help me write tests" → AI auto-selects tdd-workflow |
| **Explicit request** | "Use the api-design skill to review my endpoints" |

> AI assistants automatically match skills based on task description. You don't need to memorize trigger names — just describe what you want.
>
> AI 助手会根据任务描述自动匹配技能。你不需要记住触发名称——只需描述你想做什么。
